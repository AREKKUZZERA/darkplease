# DARK PLEASE!

Dark Please! is a browser extension that automatically generates dark themes for websites.

---

## Development

Install dependencies:

```bash
npm install
````

Run extension in debug mode:

```bash
npm run debug
```

Run tests:

```bash
npm run test
```

Run inject tests:

```bash
npm run test:inject
```

Run lint:

```bash
npm run lint
```

Build extension:

```bash
npm run build
```
